RC Input: Critical

Error Code: Settings

Have you configured your transmitter on the Input configuration page? One of the following conditions may be present:

  • An R/C input channel type must be set for each of Roll, Pitch, Yaw and Throttle. On the Input configuration page, make sure these channel inputs have a type other than "None".
  • One or more of the R/C input channel mappings are invalid.
  • The driver is uninitialized for one or more of the R/C input channels.
