System Configuration: Error

This reflects a number of configuration errors could lead to a flight error. Check SystemAlarms.ConfigError for more detail. Possible problems include:

  • Stabilization: Inappropriate stabilization mode selected, such as manual on a multirotor.
  • Multirotor: One or more axes set to not stabilize.
  • Autotune: Autotune flight mode or stabilization mode selected without enabling the module
  • AltitudeHold: Altitude hold flight mode available without the altitude hold module running
  • PositionHold: Position hold flight mode available without a path follower running
  • PathPlanner: Path planner flight mode available without a path planner running
