00031 /*
00032  * @note This is a stripped-down ADC driver intended primarily for sampling
00033  * voltage and current values.  Samples are averaged over the period between
00034  * fetches so that relatively accurate measurements can be obtained without
00035  * forcing higher-level logic to poll aggressively.
00036  *
00037  * @todo This module needs more work to be more generally useful.  It should
00038  * almost certainly grow callback support so that e.g. voltage and current readings
00039  * can be shipped out for coulomb counting purposes.  The F1xx interface presumes
00040  * use with analog sensors, but that implementation largely dominates the ADC
00041  * resources.  Rather than commit to a new API without a defined use case, we
00042  * should stick to our lightweight subset until we have a better idea of what's needed.
00043  */
