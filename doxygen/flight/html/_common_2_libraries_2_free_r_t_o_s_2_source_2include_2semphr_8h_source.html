293 /*
294  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
295  *
296  * The source code that implements the alternative (Alt) API is much
297  * simpler because it executes everything from within a critical section.
298  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
299  * preferred fully featured API too. The fully featured API has more
300  * complex code that takes longer to execute, but makes much less use of
301  * critical sections. Therefore the alternative API sacrifices interrupt
302  * responsiveness to gain execution speed, whereas the fully featured API
303  * sacrifices execution speed to ensure better interrupt responsiveness.
304  */
305 #define xSemaphoreAltTake( xSemaphore, xBlockTime ) xQueueAltGenericReceive( ( xQueueHandle ) ( xSemaphore ), NULL, ( xBlockTime ), pdFALSE )
