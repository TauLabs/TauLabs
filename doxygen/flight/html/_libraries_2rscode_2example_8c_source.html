00024  * This same code demonstrates the use of the encodier and 
00025  * decoder/error-correction routines. 
00026  *
00027  * We are assuming we have at least four bytes of parity (NPAR >= 4).
00028  * 
00029  * This gives us the ability to correct up to two errors, or 
00030  * four erasures. 
00031  *
00032  * In general, with E errors, and K erasures, you will need
00033  * 2E + K bytes of parity to be able to correct the codeword
00034  * back to recover the original message data.
00035  *
00036  * You could say that each error 'consumes' two bytes of the parity,
00037  * whereas each erasure 'consumes' one byte.
00038  *
00039  * Thus, as demonstrated below, we can inject one error (location unknown)
00040  * and two erasures (with their locations specified) and the 
00041  * error-correction routine will be able to correct the codeword
00042  * back to the original message.
00043  * */
