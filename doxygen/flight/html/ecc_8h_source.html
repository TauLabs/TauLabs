00029   Below is NPAR, the only compile-time parameter you should have to
00030   modify.
00031   
00032   It is the number of parity bytes which will be appended to
00033   your data to create a codeword.
00034 
00035   Note that the maximum codeword size is 255, so the
00036   sum of your message length plus parity should be less than
00037   or equal to this maximum limit.
00038 
00039   In practice, you will get slooow error correction and decoding
00040   if you use more than a reasonably small number of parity bytes.
00041   (say, 10 or 20)
