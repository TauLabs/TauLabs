\-Contains the information of the plugins xml description file and information about the plugin's current state.

\-Invalid \-Starting point\-: \-Even the xml description file was not read.  \-Read \-The xml description file has been successfully read, and its information is available via the \hyperlink{class_extension_system_1_1_plugin_spec}{\-Plugin\-Spec}.  \-Resolved \-The dependencies given in the description file have been successfully found, and are available via the \hyperlink{class_extension_system_1_1_plugin_spec_aea749392b5e9b90bf580fae85b956e12}{dependency\-Specs()} method.  \-Loaded \-The plugin's library is loaded and the plugin instance created (available through \hyperlink{class_extension_system_1_1_plugin_spec_ae6e82e9c2548b309eafbadfffffd4cc6}{plugin()}).  \-Initialized \-The plugin instance's \hyperlink{class_extension_system_1_1_i_plugin_a9f3dc957317279daed393fdf4894e944}{\-I\-Plugin\-::initialize()} method has been called and returned a success value.  \-Running \-The plugin's dependencies are successfully initialized and extensions\-Initialized has been called. \-The loading process is complete.  \-Stopped \-The plugin has been shut down, i.\-e. the plugin's \hyperlink{class_extension_system_1_1_i_plugin_a65c16ebd3a6e89918a99af6515fe68cb}{\-I\-Plugin\-::shutdown()} method has been called.  \-Deleted \-The plugin instance has been deleted. \begin{Desc}
