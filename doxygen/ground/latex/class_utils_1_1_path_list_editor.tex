\subsection{\-Detailed \-Description}
\-A control that let's the user edit a list of (directory) paths using the platform separator (';','\-:'). \-Typically used for path lists controlled by environment variables, such as \-P\-A\-T\-H. \-It is based on a \-Q\-Plain\-Text\-Edit as it should allow for convenient editing and non-\/directory type elements like \char`\"{}etc/mydir1\-:\$\-S\-P\-E\-C\-I\-A\-L\-\_\-\-S\-Y\-N\-T\-A\-X\-:/etc/mydir2\char`\"{}. \-When pasting text into it, the platform separator will be replaced by new line characters for convenience.
