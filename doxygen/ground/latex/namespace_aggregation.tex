\subsection{\-Detailed \-Description}
\-The \hyperlink{namespace_aggregation}{\-Aggregation} namespace contains support for bundling related components, such that each component exposes the properties and behavior of the other components to the outside. \-Components that are bundled to an \hyperlink{class_aggregation_1_1_aggregate}{\-Aggregate} can be \char`\"{}cast\char`\"{} to each other and have a coupled life cycle. \-See the documentation of \hyperlink{class_aggregation_1_1_aggregate}{\-Aggregation\-::\-Aggregate} for details and examples.
