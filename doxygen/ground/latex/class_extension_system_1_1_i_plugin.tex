\subsection{\-Detailed \-Description}
\-Base class for all plugins. 

\-The \hyperlink{class_extension_system_1_1_i_plugin}{\-I\-Plugin} class is an abstract class that must be implemented once for each plugin. \-A plugin consists of two parts\-: \-A description file, and a library that at least contains the \hyperlink{class_extension_system_1_1_i_plugin}{\-I\-Plugin} implementation.
