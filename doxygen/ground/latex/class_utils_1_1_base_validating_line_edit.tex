\subsection{\-Detailed \-Description}
\-Base class for validating line edits that performs validation in a virtual \hyperlink{class_utils_1_1_base_validating_line_edit_acaf01e0ff24a7e62f45d7a4eddb04941}{validate()} function to be implemented in derived classes. \-When invalid, the text color will turn red and a tooltip will contain the error message. \-This approach is less intrusive than a \-Q\-Validator which will prevent the user from entering certain characters.

\-The widget has a concept of an \char`\"{}initial\-Text\char`\"{} which can be something like \char`\"{}$<$\-Enter name here$>$\char`\"{}. \-This results in state '\-Displaying\-Initial\-Text', which is not valid, but is not marked red.
