Detailed Description

Impliments a HID USB connection to the flight hardware as a QIODevice.

Implements the USB monitor on Windows using system API.

Implements the USB monitor on Mac using XXXXX.

Implements the USB monitor on Linux using libudev.

Monitors the USB bus for devince insertion/removal.
