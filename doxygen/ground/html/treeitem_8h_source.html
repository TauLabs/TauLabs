00045 /*
00046 * Small utility class that handles the higlighting of
00047 * tree grid items.
00048 * Basicly it maintains all items due to be restored to
00049 * non highlighted state in a linked list.
00050 * A timer traverses this list periodically to find out
00051 * if any of the items should be restored. All items are
00052 * updated withan expiration timestamp when they expires.
00053 * An item that is beeing restored is removed from the
00054 * list and its removeHighlight() method is called. Items
00055 * that are not expired are left in the list til next time.
00056 * Items that are updated during the expiration time are
00057 * left untouched in the list. This reduces unwanted emits
00058 * of signals to the repaint/update function.
00059 */
