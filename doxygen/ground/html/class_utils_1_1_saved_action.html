QAction * SavedAction::updatedAction ( const QString &  text) [virtual]

Adjust the text() of the underlying action.

This can be used to update the item shortly before e.g. a menu is shown.

If the item's textPattern() is empty the text will be used verbatim.

Otherwise, the behaviour depends on text: if it is non-empty, QString(textPattern()).arg(text), otherwise, textPattern() with the "%1" placeholder removed will be used.

See also:
textPattern(), setTextPattern()
